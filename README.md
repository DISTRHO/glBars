# DISTRHO glBars

This is an OpenGL bars visualization plugin (as seen in XMMS and XBMC/Kodi).<br/>
Adapted from the jack_glbars project by Nedko Arnaudov.<br/>
It works as a JACK standalone and audio plugin (LV2 or VST). <br/>
